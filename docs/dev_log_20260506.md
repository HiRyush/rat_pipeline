# Secondary Pipeline — Imputation as Filter Approach (2026-05-06)

## 배경

### 연구 목적
> **DNA 없이 RNA-seq만으로 somatic mutation을 탐지하는 method 개발**
> - PHMG_IT/mammary_cancer: DNA matched → validation
> - PHMG_IH: RNA-only → final application

### 이전 시도 결과

#### 실패: Imputation-Discordance 방법 (2026-05-06 오전)
```
Imputation → Discordance(imputed REF + observed ALT) → Treatment-only
→ 117 candidates → DNA validation: TRUE SOMATIC = 0 (0%)
→ 83.8% germline leak, 16.2% RNA-only artifact
```
**실패 원인:** Imputation precision 75-82%에서 18-25%의 error가 개체별로 다른 위치에서 발생 → "Treatment-only discordance"로 위장 → somatic과 구별 불가. Imputation을 "detector"로 사용하면 작동하지 않음.

#### 성공: Phase 11 교집합 방법 (기존 결과)
```
Phase 7 (direct observation differential) ∩ Imputed differential
→ 977 candidates → DNA validation: 597 TRUE SOMATIC (61.1% PPV)
```
**성공 이유:** Direct observation이 1차 필터, imputation이 2차 확인. 두 독립 method가 동시에 동의해야 통과 → imputation error만으로는 통과 불가.

### 핵심 교훈
> **Imputation은 "detector"가 아니라 "filter"로 써야 한다.**
> - 순서: Observation → Treatment-only → Imputation confirms non-germline
> - Imputation 역할: (1) 72% gene-level coverage (2) germline 확인 필터

---

## 현재 Pipeline: Observation-first + Imputation-filter

### Pipeline 구조

```
Step 1: RNA-seq variant calling (direct observation)
        → MuTect2 force-call PASS, 15 samples
        → 기존 결과 재활용: results/mutect2/force_filtered/

Step 2: Treatment-only differential (Control vs Treatment)
        → MuTect2 Phase 9 differential: 65,375 SNPs + 19,498 INDELs
        → 기존 결과: results/mutect2/differential/

Step 3: Imputation (BEAGLE + HRDP)
        → Per-sample germline estimation, 72% gene-level coverage
        → 기존 결과: imputation/imputed/

Step 4: Intersection filter (핵심)
        → Step 2 candidates ∩ Imputation-based differential
        → "Direct observation이 Treatment-only + Imputation도 비정상"
        → 두 독립 method 동의 = high confidence

Step 5: Additional filters (Phase 11 대비 개선)
        → RNA editing (A>G/T>C) 제거
        → DESeq2 expression-aware filter (newly expressed gene 제거)
        → Population AF filter (HRDP known germline 제거)
        → GP confidence filter (imputation 불확실 site 제거)
        → DP/AF quality filter

Step 6: DNA ground truth validation
        → Sensitivity, PPV, specificity 산출
        → Per-filter-stage 성능 추적

Step 7: Post-hoc annotation
        → SnpEff + COSMIC/ClinVar enrichment
        → Mutational signature analysis (가능 시)
```

### Phase 11 대비 개선점

| 항목 | Phase 11 (기존) | 현재 pipeline |
|------|----------------|--------------|
| Observation source | Phase 7 bcftools (48K SNPs) | MuTect2 Phase 9 (65K SNPs, +34%) |
| Intersection | 단순 position overlap | Position + allele match |
| RNA editing filter | 없음 | A>G/T>C 제거 |
| Expression filter | 없음 | DESeq2 newly expressed gene 제거 |
| Population filter | 없음 | HRDP AF filter |
| GP confidence | 없음 | Imputation DS threshold |
| AF/DP filter | 없음 | Observed AF/DP quality |

### 기대 결과

| 항목 | Phase 11 | 기대 |
|------|:---:|:---:|
| Gene-level coverage | 72% | 72% (동일) |
| Somatic candidates (before filter) | 977 | 더 많음 (MuTect2 base) |
| True somatic (DNA validated) | 597 | 597+ |
| PPV | 61.1% | 상향 (추가 filter) |

Discordance 기반 (imputed REF + observed ALT → Treatment-only):

- 원인: imputation error가 개체별로 다르게 발생 → Treatment-only로 위장

### 실행 2: Observation-first + Imputation-as-Filter (성공)

#### Filter Cascade

| Stage | Count |

#### DNA Ground Truth Validation

| Stage | Total | True Somatic | Germline Leak | RNA-only | PPV |

| Step 4: MuTect2 ∩ Imputed | 1,356 | **934 (68.9%)** | 377 (27.8%) | 45 (3.3%) | **68.9%** |

| Step 5d: All filters (HRDP+AF) | 4 | 1 (25%) | 1 (25%) | 2 (50%) | 25% |

#### HRDP Filter 문제

HRDP population filter (11.3M variants)가 너무 공격적:
- 961 → 8로 삭감 (99.2% 제거)
- True somatic 657개 중 656개를 제거
- 원인: HRDP panel이 rat common variant를 거의 전부 포함 → somatic variant position이 우연히 HRDP germline site와 position 겹침 (allele는 다를 수 있으나 position 기반 매칭)
- **결론: HRDP population filter는 이 pipeline에서 제거해야 함**

#### 최적 결과 (Step 5a: MuTect2 ∩ Imputed - RNA editing)

| 항목 | Phase 11 (기존) | 현재 pipeline | 변화 |
|------|:-:|:-:|:-:|
| Total candidates | 977 | **961** | -1.6% |

**개선 요약:**
- True somatic +60개 확보 (MuTect2가 bcftools보다 감도 높음)
- RNA-only artifact 대폭 감소 (13.3% → 3.0%, RNA editing filter 효과)
- PPV 7.3%p 상승

### 남은 과제

1. **Germline leak 28.6% (275개) 저감**: 추가 filter 또는 더 정밀한 intersection 기준 필요
2. **DESeq2 expression-aware filter 적용**: gene-level mapping 후 newly expressed gene 제거 → germline leak 감소 예상
3. **mammary_cancer cross-validation**: 같은 pipeline 적용하여 generalizability 확인
4. **COSMIC enrichment analysis**: 657개 true somatic에 대해 cancer type enrichment
5. **Mutational signature**: DNA-confirmed somatic에서 PHMG signature 추출
6. **PHMG_IH 적용**: Validated pipeline으로 RNA-only dataset 분석
